# Catalina Garage

Retro-styled auto repair shop for Tucson, AZ. Old-school service, modern diagnostics. Cars, trucks, SUVs.

**Live:** https://catalina-garage.vercel.app

## Tech Stack

- Next.js 14 (App Router)
- Tailwind CSS
- Supabase (leads DB)
- Resend (email notifications)
